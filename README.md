# PR.1 Descriptive Booster – Data Analysis Project

## Overview
This project focuses on descriptive statistics using a household dataset. The dataset is generated using Python and analyzed to understand data distribution, central tendency, and variability.

## Dataset Description
The dataset contains 100 records with the following columns:

- Household_ID: Unique identifier for each household  
- Age_of_Household_Head: Age of the household head  
- Household_Income: Monthly income of the household  
- Education_Level: Primary, Secondary, Graduate, Post-Graduate  
- Family_Size: Number of family members  
- Owns_House: Yes / No  
- Urban_Rural: Urban / Rural  

## Objectives
- Identify categorical and numerical data  
- Calculate mean, median, and mode  
- Calculate range, variance, standard deviation, and IQR  
- Analyze income distribution  
- Calculate skewness and kurtosis  
- Visualize data using plots  

## Tools Used
- Python  
- pandas  
- numpy  
- matplotlib  
- seaborn  
- scipy  

## Steps Performed
1. Loaded dataset using pandas  
2. Identified categorical and numerical columns  
3. Calculated central tendency (mean, median, mode)  
4. Calculated dispersion (range, variance, standard deviation, IQR)  
5. Plotted histogram and KDE for income  
6. Fitted Gaussian distribution curve  
7. Calculated skewness and kurtosis  
8. Created boxplots for comparison
